Das <canvas> Element

Auf HTML5 Seiten kann mit einem <canvas>-Element eine Zeichenfläche (canvas, "Leinwand") definiert werden. Das ist ein Bildschirmbereich, in den mit JavaScript grafische Elemente (Linien, Rechtecke, Kreise, usw.) gezeichnet werden können. Die grafischen Elemente können mit JavaScript auch interaktive Grafiken realisiert werden.

Alle Browser unterstützen das <canvas> Element.

Die Syntax des <canvas> Elements

<canvas id="leinwand" width="400" height="100"></canvas>
<script type="text/javascript">
    var c = document.getElementById("leinwand");  // Zeiger auf Canvas
    var context=c.getContext("2d");     // Zeichnen in 2D
    context.moveTo(40,50);              // Zeichenstift positionieren
    context.lineTo(240,50);             // Linie zeichnen
    context.stroke();                   // und auf dem Bildschirm anzeigen
    context.fillStyle = "yellow";       // Füllfarbe
    context.fillRect(120,40,80,60);     // Rechteck zeichnen
    context.beginPath();                // Pfad zeichnen
    context.arc(95,50,40,0,2*Math.PI);  // in diesem Fall einen Kreis
    context.stroke();                   // und auf dem Bildschirm anzeigen
</script>

Die Alternative: Scalable Vector Graphics SVG

HTML5 ermöglicht auch das Einbinden von SVG-Grafiken. Die Scalable Vector Graphics sind eine Alternative zu den auf dem <canvas>-Element aufbauenden Grafiken.

SVG werden wir im Modul "Multimediakommunikation" eingehend behandeln.

Die Syntax von SVG

Der folgende SVG-Text beschreibt die gleiche Grafik wie oben: 

<svg width="580" height="400" xmlns="http://www.w3.org/2000/svg">
    <line x1="40" y1="50" x2="240" y2="50" stroke="black" stroke-width="1"/> 
    <rect x="120" y="40" width="80" height="60" fill="yellow"/>
    <circle cx="95" cy="50" r="40" stroke="black" stroke-width="1" fill="none"/>    
</svg>
