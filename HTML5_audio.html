Das <audio> Element

Wie für Videos  wurde bisher auch für das Abspielen von Audioinhalten auf Webseiten jeweils ein Plug-In (z.B. der Flash-Player) benötigt. Mit dem <audio> Element können Audioinhalte nun in HTML5 direkt im Browser abgespielt werden.

Quelle: Feldlerche, Vogelwarte Sempach (http://www.vogelwarte.ch/feldlerche.html)

Leider unterstützen die Browser (Apple Safari, Google Chrome, Mozilla Firefox, usw.) unterschiedliche Audio-Formate, weshalb Audioinhalte jeweils in zwei verschiedenen Formaten eingebunden werden müssen:

Das Format MP3 (MPEG 1 Audio Layer 3) unterstützen Internet Explorer (ab Version 9), Google Chrome (ab Version 6) und Safari (ab Version 5)

Das Format Ogg (Ogg Vorbis) unterstützen Firefox (ab Version 3.6), Google Chrome (ab Version 6) und Opera (ab Version 10.5).

Die Syntax des <audio> Elements

<audio controls="controls" preload="none">
   <source src="Feldlerche.ogg" type="audio/ogg">
   <source src="Feldlerche.mp3" type="audio/mpeg">
</audio>
