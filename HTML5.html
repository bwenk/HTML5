HTML5

Die neuste Version der Hypertext Markup Language HTML

HTML5 löst langsam die aktuelle Version von HTML (4.01) aus dem Jahre 1999 ab. Das World Wide Web Consortium (W3C) publizierte 2008 erstmals einen Entwurf von HTML5.

Am 17.Februar 2014 war auf der Website des WWW Consortium der "working draft" vom 25. Mai 2011 zu finden. Die Standardisierungsarbeit ist noch im Gang. Der aktuelle "editor's draft" vom 17. Februar 2014 beschreibt die Version 5.1 von HTML.

HTML5, CSS3 (Cascading Style Sheets), JavaScript und neuen JavaScript APIs (Application Programming Interfaces) bilden die Grundlagen für die Realisierung interaktiver und sogar offline nutzbarer Webapplikationen.

Alle aktuellen Versionen der gängigen Browser (Firefox, Chrome, Safari, Opera, usw.) unterstützen HTML5 bereits, allerdings nicht alle auf völlig gleiche Weise.

Die Struktur eines HTML5 Dokuments

<!DOCTYPE html>
<html>
   <head>
   </head>
   <body>
   </body>
</html>

Was bietet HTML5 im Vergleich zu HTML 4?

  • Die semantischen Elemente zur Strukturierung der HTML5-Dokumente
  • Drag & Drop
  • Das <input> Element für Benutzereingaben
  • CSS3 - die Version 3 der Cascading Style Sheets
  • Das <audio> Element zum Einbinden von Audioinhalten
  • Das <video> Element zum Einbinden von Videos
  • Das <canvas> Element zum Zeichnen von (interaktiven) Grafiken
  • localStorage - Das API zur lokalen Speicherung von Schlüssel/Werte-Paaren
  • geoLocation - Das API zur Bestimmung der geografischen Position
